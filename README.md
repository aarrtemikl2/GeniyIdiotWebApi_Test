# GeniyIdiotWebApi
Реализация первого учебного проекта с курса "PRO C#. ООП на практике" игры "Гений идиот" как WebApi
Цель - первично пощупать .Net и EFC и в целом работу с Rest. Выглядит скорее всего всё вырвиглазно
База данных SqLite
+ Пытался в архитектуру проекта

Включает в себя следующие методы:
1. Получение списка вопросов
2. Получение перемешанного списка вопросов
3. Получение рассчитаного  результата игры
4. Добавление/Удаление/Получение вопроса (в процессе)
5. Создание пользователя
